# E-Commerce Website
# Overview 
ShiriCloth is a dynamic, user-friendly e-commerce platform designed to provide a seamless online shopping experience for clothing enthusiasts. 
Focused on modern, fashionable attire, ShiriCloth offers a diverse range of products including dresses, suits, casual wear, and accessories for all genders and ages. 
This repository contains the entire codebase for the ShiriCloth website, including frontend design, backend functionality, and database management.
